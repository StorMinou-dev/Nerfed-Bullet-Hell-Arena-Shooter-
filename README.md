[Title Image]


**Gameplay**  
This game is a 3D top down arena shooter made in Unity,   
it features elements from bullet hell type of games like   
Touhou Project (which is heavily inspired of)  
The goal is to survive waves of enemies by dodging their   
projectiles and shooting them with your aim.

[Gameplay Image]

You start with 4 lives and the game is over when you   
lose all of them.

**Characters**  
The game features 3 different characters :  
[3 perso image]

The first one shoots quick daggers that disappear when hitting and enemy and do medium damage.  
[image gameplay 1er]

The second one fires slow axes that pierce through enemies, it does less damage to keep it balanced.  
[image 2]

And the third one shoots swords with a really weak fire rate but does huge damage.
